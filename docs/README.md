# Ship Deploy PoC - Architecture Diagram

This directory contains architecture diagrams and documentation images.

## Generating the Architecture Diagram

The Mermaid diagram from README.md can be rendered using:
- GitHub's built-in Mermaid support
- [Mermaid Live Editor](https://mermaid.live/)
- VS Code Mermaid extension

To create a PNG:
1. Copy the Mermaid code from README.md
2. Paste into Mermaid Live Editor
3. Export as PNG
4. Save as `architecture.png` in this directory
